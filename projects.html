1) Keylogger using Python

For this project, I created a simple keylogger to learn how keystroke logging works at a basic level. I used Python libraries to capture and record keyboard input, saving the data into a file. This project helped me understand how such tools are built, the potential risks they pose, and the importance of cybersecurity measures to defend against them.

2) Website using HTML & CSS

In this project, I built a multi-page website using HTML and CSS. I focused on structuring content clearly with HTML and adding style and layout through CSS. This allowed me to practice the fundamentals of front-end development, experiment with responsive design, and gain a better understanding of how websites are created from the ground up.

3) Packet Sniffing with Wireshark

Used Wireshark to capture and analyze network traffic. Learned how to filter packets, identify protocols in use, and detect potential vulnerabilities. This project strengthened my understanding of how data moves across a network and how to spot unusual activity.

4) Home Network Setup & Configuration

Configured a small home network with a router and switch. Assigned IP addresses, set up DHCP, and tested connectivity across multiple devices. Secured the network with WPA2 encryption and practiced basic troubleshooting.

5) Virtual Private Network (VPN) Lab

Set up a simple VPN connection on a virtual machine. Tested remote access and encryption to see how VPNs secure communication over the internet.
